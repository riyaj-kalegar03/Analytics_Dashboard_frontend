## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:5173 in your browser

## Building for Production

1. Build the project:
   ```bash
   npm run build
   ```
2. Preview the build:
   ```bash
   npm run preview
   ```
